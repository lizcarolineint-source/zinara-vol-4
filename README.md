# Zinara Digital — Images Folder

## How to replace any image

1. Prepare your photo — compress at squoosh.app (keep under 400KB)
2. Name it EXACTLY as shown in the table below
3. Upload it to this images/ folder on GitHub
4. Update the src in the HTML from .svg to .jpg
5. Vercel auto-deploys in under 60 seconds

## How to swap placeholder for real photo

In the HTML file, change the image reference from .svg to .jpg.

Example in index.html:
  Before: src="images/problem-team.svg"
  After:  src="images/problem-team.jpg"

Edit directly in GitHub (click file, click pencil icon, commit).
Vercel deploys automatically.

---

## Tips

- Landscape orientation (wider than tall) works best for all slots
- Compress at squoosh.app before uploading — target under 300KB
- Hero background looks best dark (city at night, office) since a navy overlay sits on top
- Format: JPEG for photos, PNG for logos and graphics
